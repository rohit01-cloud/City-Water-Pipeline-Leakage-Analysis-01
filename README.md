#  City Water Pipeline Leakage Analysis

This project analyzes city water pipeline data to detect possible water leakages using statistical analysis and AI concepts.

##  Features
- Water loss calculation
- Z-score based leakage detection
- Data visualization using graphs
- Web-based interface using Gradio

##  Technologies Used
- Python
- Pandas, NumPy
- Matplotlib
- SciPy
- Gradio
